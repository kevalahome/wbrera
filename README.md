# 🏠 Kevala Home — WB RERA Check

**Free, offline-first PWA to instantly verify West Bengal RERA registered real estate projects.**

🔍 Search 4,250+ projects | 📱 Install as app | 🌐 Works offline | 🛡️ Official data

---

## ✨ Features

- **Instant Search** — 4,250+ projects by name or RERA number
- **Offline-First PWA** — Full functionality without internet
- **Installable** — Add to home screen on Android & iOS
- **RERA Verification** — Direct link to official WB RERA portal
- **Safety Checklist** — 10-point buyer protection guide
- **Dark/Light Mode** — Auto-adapts to system preference
- **Bookmarks** — Save projects for quick reference
- **Share** — Send project details via WhatsApp
- **Zero Analytics** — No tracking, 100% private
- **Tiny & Fast** — Single HTML file, loads instantly

## 🚀 Quick Start

### Host on GitHub Pages (Recommended)

1. **Fork this repository** to your GitHub account
2. Go to **Settings → Pages**
3. Select branch: `main`, folder: `/ (root)`
4. Click **Save**
5. Your app is live at: `https://<your-username>.github.io/<repo-name>/`

### Run Locally

```bash
python3 -m http.server 8000
# Open http://localhost:8000 in your browser
